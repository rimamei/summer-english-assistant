# Changelog

All notable changes to Summer English AI will be documented in this file.

## 0.0.1 (2025-11-21) - Beta Release

Welcome to the first beta release of Summer English AI! This Chrome extension helps you learn English with AI-powered tools.

### Features

#### Translation
- Text translation with support for multiple languages
- Dual AI engine support: Chrome Built-in AI and Google Gemini
- Select text on any webpage to translate instantly
- Full-page translation support
- Visual progress indicators for AI model downloads

#### Grammar Analysis
- AI-powered grammar checking and correction
- Detailed explanations of grammar sentence
- Markdown-formatted output for better readability
- Support for both Chrome AI Built-In and Gemini models

#### Text Summarization
- Summarize long articles or selected text
- Customizable summary type and length
- Multiple summarization modes

#### Vocabulary & Pronunciation
- Vocabulary learning with translations
- Text-to-speech pronunciation support
- Audio playback for learning proper pronunciation
- Accent selection for different English variants

#### User Interface
- Floating draggable modal for results
- Dark mode support across all UI components
- Clean, modern interface with Inter font
- Smooth animations and transitions
- Click-outside-to-close functionality

#### Configuration & Settings
- Comprehensive preferences and configuration forms
- Extension enable/disable toggle
- Language preferences (source and target languages)
- Theme selection (light/dark mode)
- Audio and microphone testing
- Persistent settings with local storage sync

#### Internationalization
- Multi-language UI support (i18n)
- Localized content for modal and popup

### Developer Experience
- Add test coverage with Vitest
- ESLint, Prettier, Commitlint integration
- Husky pre-commit hooks with build validation
- Hot Module Replacement (HMR) for faster development
- Improved build performance
- Complete documentation and README

### Performance
- Lazy loading for result modals
- Optimized component rendering
- Improved error handling and error boundaries
- Efficient state management
